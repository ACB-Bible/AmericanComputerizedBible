American Computerized Bible

American Computerized Bible
is a computerized Bible platform and it's source code is the wholly owned property of ACB-Bible that was begun in 2019 and was first published in 2023.

Twenty-First Century Bible
is a translation of the Holy Bible which is part of the American Computerized Bible, and it's the wholly owned property of ACB-Bible. Translation of the Twenty-First Century Bible was begun in 2019 and was first published in 2023.

These properties include, but are not limited to all the source code files, css files, javascript files, html files, text files, sql database files, and Json files. The American Computerized Bible and the Twenty-First Century Bible are both incomplete at this time, and work on both projects will continue for some time.
